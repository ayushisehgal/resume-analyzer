# 🚀 AI Resume Analyzer

An **intelligent web application** that analyzes resumes using **Artificial Intelligence (AI)** and provides **detailed insights, skill evaluation, and improvement suggestions**.

---

## 🔥 Features

✨ Upload resume (PDF format)
✨ Extracts text using **Apache PDFBox**
✨ AI-powered resume analysis using **OpenRouter API**
✨ Identifies:

* 🧠 Skills
* 💼 Strengths
* ⚠️ Weaknesses
* 📈 Improvement Suggestions

✨ Clean and simple **React UI**
✨ Backend powered by **Spring Boot**

---

## 🛠️ Tech Stack

### 💻 Frontend

* React.js
* HTML5 / CSS3
* Fetch API

### ⚙️ Backend

* Java
* Spring Boot
* REST APIs

### 🤖 AI Integration

* OpenRouter API (GPT-based models)

### 📄 Resume Processing

* Apache PDFBox

---

## ⚡ How It Works

1. User uploads resume (PDF)
2. Backend extracts text from file
3. Text is sent to AI model
4. AI analyzes resume and returns structured insights
5. Results displayed on frontend

---

## ▶️ How to Run

### 🔧 Backend (Spring Boot)

```bash
mvn spring-boot:run
```

Runs on:

```
http://localhost:8080
```

---

### 🌐 Frontend (React)

```bash
cd resume-ui
npm install
npm start
```

Runs on:

```
http://localhost:3000
```
